Adding Big Number

Kata task

We need to sum big numbers and we require your help.Write a function that returns the sum of two numbers. The input numbers are strings and the function must return a string.

Example

  • add("123", "321"); -> "444"
  • add("11", "99"); -> "110"
  • add('63829983432984289347293874', '90938498237058927340892374089'); -> "91002328220491911630239667963"

Notes

  • The input numbers are big.
  • The input is a string of only digits.
  • The numbers are positives.

Solution of task:

		function add(a, b) {
			a = [...a].reverse()
			b = [...b].reverse()
		
			for (let i = 0; i < b.length; i++) {
				a[i] = ~~a[i] + ~~b[i];
				if (a[i] > 9) {
					a[i] -= 10
					b[i + 1] = ~~b[i + 1] + 1
				}
			}
		
			return a.reverse().join("")
		}
